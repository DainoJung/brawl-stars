# 👵 노인 친화형 약 복용 관리 앱 (Elderly Medicine Reminder)

> “사진 한 장으로 약을 등록하고, 복용 시간을 자동으로 챙겨주는 노인 맞춤형 약 복용 관리 앱”

---

## 📘 프로젝트 개요

### 🎯 목적
노인 사용자들이 매일 복용해야 하는 약을 **쉽고 안전하게 관리**할 수 있도록,
사진 기반 자동 인식(OCR)과 강력한 알림, 보호자 알림 기능을 결합한 **약 복용 관리 솔루션**입니다.

### 🧩 주요 타깃
- 60~80대 노인
- 부모님 약을 대신 챙겨주는 가족, 보호자
- 요양시설 관계자

### 💡 핵심 가치
- **입력 최소화:** 약 봉투 촬영만으로 자동 인식  
- **복용 안전성:** 상호작용 및 중복 복용 경고  
- **복용 지속성:** 강한 알림 + 보호자 알림으로 복용 누락 방지  

---

## 🧭 사용자 시나리오

1. 사용자는 약 봉투를 촬영하거나 직접 검색해 약을 추가  
2. 앱은 OCR을 통해 약명·용량·복용 횟수를 자동 인식  
3. 자동 스케줄 추천 → 복용 시간 알람 등록  
4. 알람 발생 시 전체화면 알림 + 사진 인증으로 복용 확인  
5. 복용 누락 시 보호자에게 알림 전송  
6. 챗봇을 통해 약 관련 질문 (“식전인가요?”, “부작용 있나요?” 등) 즉시 답변

## 📱 주요 기능 요약

### 🏠 홈
> 오늘 복용 현황과 다음 알림을 한눈에 확인하고, 주요 기능으로 바로 이동

| 구분 | 기능 | 설명 |
| --- | --- | --- |
| 복용 현황 | 오늘의 약 리스트 | ‘복용 완료 / 예정 / 누락’ 상태별 카드 표시 |
| 다음 알림 | 다음 알림 시간 표시 | “다음 알림: 오후 8시” 형식 |
| 빠른 실행 카드 | 📷 약 추가 / 🤖 챗봇 / 👪 보호자 보기 | 원클릭 이동 가능한 카드 UI |

### 💊 약 관리
> 약 추가, 수정, 삭제 및 세부 정보 관리

| 구분 | 기능 | 설명 |
| --- | --- | --- |
| 약 추가 | 사진으로 추가 | 카메라로 약 봉투 촬영 → OCR 분석 |
|  | 직접 검색 | 약 이름 입력 후 선택 |
|  | 알람 시간 자동 추천 | 복용 횟수에 따라 자동 시간 배정 |
|  | 수동 수정 | 사용자가 직접 시간 변경 가능 |
| OCR 결과 | 인식된 후보 약 리스트 표시 | 동일 성분 중복 시 경고 배너 |
| 세부 설정 | 용량, 복용 횟수, 식전/식후, 복용 기간 입력 |  |
| 기존 약 관리 | 내 약 리스트 보기, 수정/삭제 |  |
| 약 상세 카드 | 약명, 복용 시간, 주의 문구 표시 | 기능별 카테고라이징 |

### ⏰ 알람 관리
> 복용 알람 스케줄 설정 및 알림 소리/강도 조절

| 구분 | 기능 | 설명 |
| --- | --- | --- |
| 알람 리스트 | 약별 시간 리스트 표시 | 예: 아침 08:00, 저녁 20:00 |
| 알람 소리 | 점점 커지는 알림음 옵션 | 시각·청각 장애 고려 |
| 전체화면 알람 | 복용 시점에 전체화면 팝업 + 진동 | 약 복용을 놓치지 않게 함 |
| 복용 인증 | 사진으로 복용 완료 인증 |  |
| 알림 액션 | “복용 / 10분 후 / 건너뛰기” 버튼 제공 |  |
| 미루기(스누즈) | 일정 시간 후 재알림 |  |
| 누락 처리 | 건너뛰면 보호자에게 알림 발송 |  |
| 알람 수정 | 시간 직접 변경 가능 |  |
